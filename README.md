# Разработка интернет-приложений (РИП) 2025
## Расчет стоимости владения активом (ТСО)
Разработанное приложение позволит аналитикам создавать и формировать заявки на расчёты ТСО с выбранными ими услугами ТСО, а проектным менеджерам завершать или отклонять данные заявки.
